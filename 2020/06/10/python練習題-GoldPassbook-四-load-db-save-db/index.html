python練習題: GoldPassbook (四) load_db, save_db

Posted on 2020-06-10 Edited on 2020-06-15 In python練習題 Views:

可以連接 MongoDB 了, 也可以從 CSV 檔匯入資料了, 再來就是把資料寫入 MongoDB 裡了…

首先, 宣告一個 save_db 的 method 來將 self.__df 的內容寫入 MongoDB

要存入 MongoDB 裡的資料格式為: { “Date”, “Buy Cost”, “Sell Cost”} (MongoDB 會自動加上 “_id”)
但 DataFrame 的資料裡 “Date” 為 index, 所以要先 reset_index 把 “Date” 還原成一欄才行

method 的做法是: 先將 MongoDB 的資料讀出來, 然後和 self. __df 的資料比對”Date”欄位,
如果有找到, 就將該筆資料更新成 self.__df 的內容
如果沒有找到, 就新增一筆資料

接下來是 load_db, 將 MongoDB 裡的資料讀出來存入 self.__df

從 MongoDB 讀取資料比較簡單, 只是要再把格式換成 index=”Date” 而已
