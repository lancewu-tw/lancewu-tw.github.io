App Icon Generator

Posted on 2020-05-28 In 免費資源 Views:

網址: https://appicon.co/

這是從 Complete Flutter App Development Bootcamp with Dart 這門課裡看到的, 一個可以免費幫忙轉換 icon 圖的網站
只要上傳一張圖, 就可以幫你轉換成 iphone 和 android 等裝置所需要的 icon 圖
(其實就是把原始圖案縮小成不同的 size 來因應不同裝置所需的 icon 圖)

因為我的是 android app, 所以就只需要 android 的 icon 圖, 按 Generate 後會下載一個 AppIcons.zip, 裡頭有很多目錄, 主要的是 android 目錄下的各個 dpi 目錄, 裡面就是放不同 dpi 裝置所需的 icon 圖
appstore.png 和 playstore.png 則是要上架 Google Play Store 所需要的圖檔, 不一定要用這裡產生的

照課程裡的做法, 是要把這些 dpi 目錄 copy 到 android 程式專案目錄下的特定位置, 但我用的是 Cordova, 所以只要修改目錄下的 config.xml, 在 android 的 tag 內加上以下的 icon 程式碼再編譯就可以了
