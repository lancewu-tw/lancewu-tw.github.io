這就是人生

Posted on 2020-05-22 Edited on 2020-05-28 In MC 二三事 Views:

原本 run 的好好的 MC + 下單大師自動啟動…在今天出門保養車子時竟然出問題了… Orz

時間到時發現沒有 email 通知就覺得怪怪的…
後來打到停損點也沒下單, 就知道糗了…趕快手動停損, 多損了一些…
下午回家後發現… 下單大師竟然沒有自動登入 @@”
查了一下原因, 不確定是否是因為 windows 自動更新造成的問題…
只好馬上做一些處置, 之後再測試看看了…

  1. 設定 windows 的自動更新時間, 避開白天的時段
  2. 安裝 teamviewer, 並設定能從平板連回電腦處理, 這樣有問題時就可以連回去看看 (不過如果是網路斷線就沒用了)

換個想法, 因為這個突發狀況把一些除錯或備援的機制弄好, 也是個好機會…
其他策略因為沒到達預定點位, 所以沒動作, 損失不大, 算不幸中的大幸 ^^”
(不過動作應該可以再快點, 發現有問題就要處理了, 還好後來有手動停損, 否則等到回家就虧更多了… Orz )
